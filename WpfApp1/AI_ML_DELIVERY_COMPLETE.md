2. **Select Folder**
   - Click "📁 Select Folder" button
   - Choose folder to analyze

3. **Generate Suggestions**
   - Click "⚡ Get Smart Suggestions"
   - AI analyzes and generates suggestions

4. **Review Results**
   - See file names, suggested categories, destinations
   - Check confidence percentages (green = high confidence)
   - Read "Reasons" column for explanation

5. **Accept/Reject**
   - Click "✓ Accept" to create rules
   - Click "✗ Reject" to skip

6. **View Rules**
   - Created rules appear in main rules grid
   - Use for file organization

## 📊 Technical Implementation

### Algorithm Complexity
```
O(n × m) where:
- n = number of files to analyze
- m = number of existing rules

Performance:
- 100 files: <500ms
- 500 files: <2s
- 1000 files: <4s
```

### Scoring Formula
```
Confidence = (Extension Match × 60%) 
		   + (Category Match × 25%) 
		   + (Keyword Match × 15%)

Example:
- PDF file analyzed
- .pdf exists in rules (100%) × 60% = 60%
- Category "Documents" matches 80% × 25% = 20%
- Keywords suggest documents 50% × 15% = 7.5%
- Total: 87.5% confidence
```

### Adaptive Learning
```
Accept Suggestion:
├─ Accuracy: +5%
├─ Confidence: +2 points
└─ Next similar suggestions more confident

Reject Suggestion:
├─ Accuracy: -3%
├─ Confidence: -1 point
└─ Next similar suggestions more cautious
```
